How it works

          
Browser-based `.npy` to `.xyz` conversion

        

        

          

            
1. Open a `.npy` file

            

              Choose a NumPy file that contains point data. The tool reads the
              file header and array structure directly in the browser.
            

          

          

            
2. Extract XYZ coordinates

            

              For numeric arrays, the first three values of each point become
              the output coordinates. Structured arrays can also use `x`, `y`,
              and `z` fields.
            

          

          

            
3. Download plain-text output

            

              The result is saved as a `.xyz` text file where each line contains
              `x y z` values, ready for point-cloud workflows and downstream
              processing.

FAQ

          
Common questions

        

        

          

            Does this upload my file to a server?
            

              No. The static version processes files locally in your browser, so
              your `.npy` file stays on your device during conversion.
            

          

          

            Can I convert `.npz` files too?
            

              Not yet. This tool is focused on `.npy` files only.
            

          

          

            Why is the output plain text?
            

              Standard `.xyz` files are text-based, so each output line contains
              one point in the form `x y z`.
            

          

          

            What if my array has more than 3 values per point?
            

              The converter uses the first three values from the last dimension
              of each point and ignores any extra trailing values.
